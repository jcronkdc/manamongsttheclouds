## Workflow

1. When starting a new editing pass, copy the latest rev and increment the number:
   ```
   Part I - The Still Water_rev01.docx  →  Part I - The Still Water_rev02.docx
   ```
2. Make all edits in the new rev file. Never overwrite a previous rev.
3. The highest `_revNN` number is always the current working version.
