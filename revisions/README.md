# Revision Tracking — Man Amongst the Clouds

## Naming Convention

All docx files follow this pattern:

```
<Title>_rev<NN>.docx
```

- **rev00** — Original baseline (pre-edit)
- **rev01** — First round of edits
- **rev02** — Second round of edits
- ...and so on

## Current Files

| File | Description |
|------|-------------|
| `Part I - The Still Water_revNN.docx` | Part I manuscript |
| `Part II - The Waking_revNN.docx` | Part II manuscript |
| `Part III - The Breaking_revNN.docx` | Part III manuscript |
| `Part IV - The Chamber_revNN.docx` | Part IV manuscript |
| `Part V - The Remembering_revNN.docx` | Part V manuscript |
| `Man Amongst the Clouds - Manuscript_revNN.docx` | Full combined manuscript |
| `Man Amongst the Clouds - Print Copy_revNN.docx` | Print-formatted copy |

## Workflow

1. When starting a new editing pass, copy the latest rev and increment the number:
   ```
   Part I - The Still Water_rev00.docx  →  Part I - The Still Water_rev01.docx
   ```
2. Make all edits in the new rev file. Never overwrite a previous rev.
3. The highest `_revNN` number is always the current working version.
